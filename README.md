# 🍕 Pizzeria App

Web application for ordering food where users can explore the menu, customize products, add items to the cart and complete the checkout process using different payment methods.

Users must be registered and logged in to complete a purchase. Once authenticated, they can select products, customize ingredients, manage their cart and finalize the order by paying with credit card, PayPal or cash at the store.

The project is built with **Next.js and React**, using **Redux for global state management** and a **mock API to simulate backend operations**.

---

## 🚀 Features

- User registration and login
- Authentication required to complete purchases
- Browse food menu
- Product customization (ingredients selection)
- Dynamic shopping cart
- Product review system
- Checkout with multiple payment methods:
  - Credit Card
  - PayPal
  - Cash at the store
- Global state management with Redux
- Responsive UI
